# CS2 Coach Bench — Scoring Rubric

## 5 Dimensions, 0-3 Each (max 15 per question, 3000 total)

### 1. Tactical Correctness

| Score | Criteria |
|-------|----------|
| 0 | Factually wrong — advice would hurt the player |
| 1 | Partially correct — right direction, wrong details |
| 2 | Mostly correct — minor inaccuracies or missing edge cases |
| 3 | Exactly right — a tier-1 coach would give the same advice |

### 2. CS2-Currentness

| Score | Criteria |
|-------|----------|
| 0 | Uses CSGO-era info (old maps, removed mechanics, outdated meta) |
| 1 | Mostly CS2 — minor CSGO-era references or outdated callouts |
| 2 | All CS2 — correct mechanics, maps, and economy system |
| 3 | References current competitive meta (2025-2026 Premier season) |

### 3. Specificity

| Score | Criteria |
|-------|----------|
| 0 | Generic platitudes ("play smarter", "aim better") |
| 1 | Some specifics — names a position or concept but no detail |
| 2 | Named callouts — references specific map positions, angles, timings |
| 3 | Concrete specifics — exact positions, tick-level timing, damage numbers |

### 4. Pro Grounding

| Score | Criteria |
|-------|----------|
| 0 | No professional references |
| 1 | Vague references ("pros do this") |
| 2 | Named pros or teams with context |
| 3 | Named pros + verifiable stats or specific match references |

### 5. Actionability

| Score | Criteria |
|-------|----------|
| 0 | Descriptive only — explains what happens, not what to do |
| 1 | One action — "do X" with no context or alternatives |
| 2 | Multi-step — "do X, then Y, and if Z happens, do W" |
| 3 | Multi-step + decision criteria — when to use which approach |

## Scoring Rules

- Score each dimension independently
- If the answer is "I don't know" or refuses to answer, score all dimensions 0
- If the answer is correct but for the wrong game version (CSGO), cap CS2-currentness at 0
- Pro grounding: fabricated stats or wrong team attributions score 0 on that dimension
- A perfect tactical answer with no pro references still gets 0 on pro grounding
