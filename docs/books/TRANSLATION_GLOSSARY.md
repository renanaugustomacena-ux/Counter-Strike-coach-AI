## Style notes

**Italian → English:**
- Formal "tu" stays as "you" (neutral, matching coach-player tone).
- Long Italian sentences with multiple subordinate clauses can be broken into 2-3 shorter English sentences for readability.
- "Ad esempio" / "per esempio" → "for example" (avoid "e.g." in flowing prose).
- "È" / "sono" often contracts to "is" / "are" in English.
- Italian hedge "in un certo senso" → "in a sense" or omit for directness.

**Italian → Portuguese (PT-BR, ASCII-only):**
- Formal "tu" (IT) → "voce" (PT, no accent) matching existing PT README convention.
- "È" (IT) → "e" (PT without accent).
- "Perché" → "porque" (without accent) or "por que" for "why".
- "Più" → "mais".
- Gender agreement follows PT rules (feminine/masculine).
- Technical terms keep English forms (see above list).

**Parity check after translation:**
1. H1 / H2 / H3 outline must match source 1:1.
2. Every file:line anchor preserved verbatim.
3. Code blocks unchanged.
4. Mermaid diagrams unchanged (node labels translated, structure identical).
5. Tables have same number of rows and columns.
6. Blockquote `> **Analogia:**` / `> **Analogy:**` / `> **Analogia:**` formatting preserved.
